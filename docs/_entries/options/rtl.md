---
title: rtl
name: options-rtl
---

Set right-to-left support (true / false). Default is false.

{% highlight js %}
$('#tree1').tree({
    rtl: true
});
{% endhighlight %}

You can also set the option using ``data-rtl``.

{% highlight html %}
    <div id="tree1" data-url="/data" data-rtl></div>
{% endhighlight %}
