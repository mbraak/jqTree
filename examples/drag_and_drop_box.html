Example 11 - Drag and drop with box dragging

Jqtree also supports a different "box" style method of drag and drop. Let's add this type of drag-and-drop support by setting the option dragAndDrop to 'box'. Additionally you will need styling that works with 'boxed' items. You can find an example in example.css . You can now drag tree nodes to another position with this box style.

html

<div id="tree1" data-url="/example_data/"></div>

javascript

var $tree = $('#tree1');
$tree.tree({
  dragAndDrop: 'box'
});
