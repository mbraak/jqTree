jqTree

JqTree is a tree widget written in jQuery UI.

Features

  • Create a tree from JSON data
  • Drag and drop
  • Works on ie7+, firefox 3.6+, chrome and safari
  • Works with jQuery UI 1.8 or 1.9 beta

The project is hosted on github, has a test suite and a demo. 

var data = [
	{
		label: 'node1',
		children: [
			{ label: 'child1' },
			{ label: 'child2' }
		]
	},
	{
		label: 'node2',
		children: [
			{ label: 'child3' }
		]
	}
];
$('#tree1').tree({
	data: data,
	autoOpen: true,
	dragAndDrop: true
});

Tree options

data

Define the contents of the tree. The data is a nested array of objects. This option is required.
It looks like this: 

var data = [
	{
		label: 'node1',
		children: [
			{ label: 'child1' },
			{ label: 'child2' }
		]
	},
	{
		label: 'node2',
		children: [
			{ label: 'child3' }
		]
	}
];
$element.tree({data: data});
  • label: label of a node (required)
  • children: array of child nodes (optional)

You can also include other data in the objects. You can later access this data.
For example, to add an id: 

{
	label: 'node1',
	id: 1
}
autoOpen

Open nodes initially.

  • true: open all nodes.
  • false (default): do nothing
  • n: open n levels

Open all nodes initially: 

$element.tree({
	data: data,
	autoOpen: true
});

saveState

Save and restore the state of the tree automatically.
Saves in a cookie which nodes are opened and selected.

  • true: save and restore state in a cookie
  • false (default): do nothing
  • string: save state and use this name to store in a cookie

Save state: 

$element.tree({
	data: data,
	saveState: true
});

Example: save state in cookie 'tree1': 

$element.tree({
	data: data,
	saveState: 'tree1'
});
dragAndDrop

Turn on dragging and dropping of nodes.

  • true: turn on drag and drop
  • false (default): do not allow drag and drop

Example: turn on drag and drop. 

$element.tree({
	data: data,
	dragAndDrop: true
});
selectable

Turn on selection of nodes.

  • true: turn on selection of nodes
  • false (default): turn off selection of nodes

Example: turn on selection of nodes. 

$element.tree({
	data: data,
	selectable: true
});
